Prework Study Guide

HTML

  • The html element has a lang attr. for setting the default website language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • id selectors start with a #
  • select by attribute with the ['attributeName']

Git

  • git pull origin main It is a good habit to always pull the base branch into your feature branch before pushing up any code.
  • To make a feature branch use the git checkout -b <branch-name>
  • git rebase overwrites commits on the branch your pulling too
  • git status: looks at the working tree of the current branch.
  • git add -A or git add . : to add all the currently untracked files to the commit
  • git commit -m "<commit-message>": the -m attaches a message of your choosing to send with the commit

JavaScript

  • Adding event listeners to buttons with <button-element>.addEventListener('<event-type>', () => {} or function())
